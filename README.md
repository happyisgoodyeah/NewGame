## 快速开始

### 环境要求

- **Unity 版本**: 2022.3.+ ~ 6000 均支持
- **IL2CPP**: 需要安装 IL2CPP 打包环境，否则 F6 编译会报错

### 运行步骤

1. 打开 Unity 项目
2. 切换到 `Packages/cn.etetet.loader/Scenes/Init` 场景
3. 按 `F6` 编译
4. 进入游戏

## 入门前言

> **YIUI 是一个偏大的 UI 框架，内容非常多，请保持耐心学习视频与文档的全部内容。**

### 学习前请确认

- YIUI **不推荐刚入门的新人学习**，特别是 ET 框架的 YIUI。如果确定要学习，请做好付出比别人更多努力的准备
- YIUI 基于 Unity UGUI，你至少需要了解 Unity 和 UGUI 相关基础知识
- 框架内部集成多个框架/插件，遇到问题请先确认问题归属：
  - ET 框架、Luban、YooAsset、HybridCLR、LoopScrollRect、I2Localization 等

### 遇到问题怎么办？

请按以下顺序尝试解决：

1. **先翻译** - 报错日志如果英文不好，先翻译理解
2. **查文档** - 查看 YIUI 文档和对应插件的官方文档
3. **找 FAQ** - 查看常见问题解答
4. **搜索** - AI 尝试解决
5. **分析** - 使用排除法定位问题

### 如何提问

如果以上方式都无法解决，在群里寻求帮助时请注意：

⚠️ **重要**: 如果实在无法解决需要向群里提问，**请让你的 AI 总结归纳后提问**，包含常规提问信息后再发群里。

❌ **禁止**: 一句话、一张图 —— 此类问题将直接无视

✅ **应该提供**:
- ET版本,YIUI版本
- 问题描述
- 已尝试的解决方案
- 期望达到的效果

> 💡 提问前强烈推荐阅读《提问的智慧》、《如何向开源社区提问题》
